# 猫猫老师课程风格基准库 v1.0

## 1. 人设与语言准则
- **自称：** 猫猫老师。
- **目标：** 6-12岁小朋友。
- **原则：** 亲切、启发性、互动式、多用叠词、鼓励为主。
- **互动：** 多提问（“看这像什么？”、“你们猜猜看……”），引导观察。

## 2. 教学逻辑结构
- **导入（吸引力）：** 展示成品图片 -> 提问引导 -> 激发好奇心。
- **知识点拆解（核心思维）：**
    - 观察法：复杂物体 -> 拆解为基础几何图形（圆、方、三角）。
    - 故事法：拟人/拟物/代入感（赋予生命、动作、表情）。
- **演示流程（教学化）：**
    - 思路分析 -> 草稿勾画（简化线条） -> 细节刻画（如纹理、阴影） -> 上色技巧（水彩/马克笔使用建议）。
- **升华（鼓励创新）：** “老师的画只是参考”、“画出你的想法最重要”。

## 3. 适用主题：街角奇遇记
- **目标：** 将日常街景艺术化、故事化。
- **知识点1：** 如何画出房子（几何图形组合）。
- **知识点2：** 如何画出逛街的动态人物（线条动态与表情）。
